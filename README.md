# Dieta do Zen - Projeto Android Real

Este ZIP contém um projeto Android real, pronto para:
- subir no GitHub;
- gerar APK real via GitHub Actions;
- instalar no Android como app de teste.

## Como gerar o APK
1. Extraia este ZIP.
2. Suba tudo para um repositório novo no GitHub.
3. Abra a aba Actions.
4. Rode **Build Android APK Real**.
5. Baixe o artefato `dieta-do-zen-apk-real`.
6. Dentro estará o `app-debug.apk`.
